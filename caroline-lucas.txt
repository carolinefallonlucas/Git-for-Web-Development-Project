    1. What is Semantic HTML? 
        Semantic html is the use of HTML markup language to convey meaning about the information on the webpage. 
    2. What is HTML used for? 
        HTML is the markup language for documents that are/will be displayed in a web browswer. 
    3. What is an attribute and where do we put it? 
        Attribues modify or add meaning to HTML elements. 
    4. What is the h1 tag used for? How many times should I use it on a page?

    7. What is an id? 
        An id is an optional, unique element attribute. ID's are useful for direct linking. 
    8. What elements can I add an id to? 
        id's can be added to any html element. 
    9. How many times can I use the same id on a page? 
        id's can be used ONCE per page. 
    10. What is a class? 
        A class is an optional, custom element attribute that can be used on multiple elements on the same page. 
    11. What elements can I add a class to? 
        A class can be added to any html element. 
    12. How many times can I use the same class on a page? 
        There is no limit on the number of times a class is used.
